固件信息：

荣耀盒子pro是hi3798mv100芯片，基于安卓5.1的系统。固件里面跟安卓内容大致相同。

解开HiTV-M1 2.1.1.68-update.bin

会得到：

META-INF
system
baseparam.img
boot.img
cust.img
fastboot.img
logo.img
pqparam.img
recovery.img
file_contexts

有些人想要去掉更新，可以删除\system\app\HwOUC\这个文件夹，就可以完全屏蔽更新，或者用adb停用或删除这个apk。

adb命令是

  adb shell pm uninstall --user 0 com.huawei.acp.hwouc

关于盒子的root：

固件已经被华为加强了root的提权限制，所以即使修改固件，也不一定能获取root权限，不过还是可以尝试一些低版本或者其他方法的。

切勿乱修改固件，固件修改不得当机器会进入“正在清除数据，请勿断电”的死循环模式。

关于刷机与救砖：

 1. 普通的刷机需要FAT32的U盘，而且不能用电脑自带的格式化功能，最好用手机格式那种软件。
 
 2. 在正常的开机页面，弹出升级提示，然后点击。
 
 3. 请不要使用返工镜像的recovery.img，它会清除你的机器信息（SN/MAC....)，机器会进入装备测试模式。
 
 4. hitool不一定能够救砖，因为没有合适的fastboot和烧录EMMC的xml表。

荣耀盒子pro常出现的问题以及解决思路：

1. 无法开机，通电灯不亮

 有些是电源问题，断电几个小时再插上电源会开机，
 不行的话，一般是硬件坏了，尤其是内存坏了，需要维修。

2. 无法开机，通电灯狂闪

 这类是硬件问题，只能尝试维修。

3. 无法开机，卡在CIBN界面

 系统出问题，需要进行刷机。

4. 无法开机，卡在honor界面

 系统出问题，需要进行刷机。

5. 无法开机，开在“正在清除信息…”界面

 系统文件异常，需要进行刷机。

6. 无法开机，卡在“装备测试”

 系统信息丢失，需要救砖刷机

7. 低版本无法观看视频。

 华为已经限制低版本的视频app的连接使用，请更新到最新版。

8. 89版本无法使用dts问题。

 这个固件有些网友反馈有问题，建议降级到85版本

9. 无法检测到升级版本。

 找华为的员工反馈，让他们推送最新版本
